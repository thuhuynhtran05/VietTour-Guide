# VietTour-Guide
VietTour Guide là hệ thống web hỗ trợ kết nối khách du lịch với hướng dẫn viên địa phương.  Hệ thống cho phép quản trị viên quản lý địa điểm, gán hướng dẫn viên cho từng tour,  người dùng có thể xem địa điểm, chọn hướng dẫn viên, đặt lịch tour và thực hiện thanh toán, xây dựng bằng Node.js, Express, MongoDB và giao diện HTML/CSS/JavaScript.
